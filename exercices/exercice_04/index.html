Exercice 04

Note : Pour voir les messages produits par le script, ouvrez la console en faisant clic droit -> inspecter, puis allez dans l'onglet "Console"

En complétant le fichier script.js :

  1. Créez un objet place qui contient les clés / valeurs suivantes :
    • clé "Name", de valeur "HEIG-VD"
    • clé "Type", de valeur "University"
    • clé "Coordinates", de valeur [2540499, 1181186]
    • clé "Address", de valeur "Route de Cheseaux 1"
    Puis affichez dans la console chaque clé est sa valeur correspondante.
  2. On vous donne la liste des cantons suisses avec quelques propriétés, sous forme d'objet (voir le fichier script). Affichez dans la console les valeurs suivantes :
    • La superficie totale de la Suisse
    • La superficie de la Suisse romande
    • Le nom du plus grand canton suisse
